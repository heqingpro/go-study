## Java 证书库添加公司根证书

Java 使用 `keytool` 命令来管理证书库（keystore），默认的证书库是 JRE 的 `cacerts` 文件。

### 方法1：使用 keytool 导入证书（推荐）

1. __定位 Java 的 cacerts 文件位置__：

```bash
# 找到你的 JDK 安装路径
echo $JAVA_HOME

# macOS 默认位置示例
/opt/homebrew/opt/openjdk/libexec/openjdk.jdk/Contents/Home/lib/security/cacerts

# 或者查看系统使用的 Java
/usr/libexec/java_home -V

# 如果不确定，可以运行：
find /Library/Java -name cacerts 2>/dev/null
```

2. __导入公司根证书__：

```bash
# 语法
sudo keytool -import -trustcacerts -keystore /path/to/cacerts \
  -storepass changeit -noprompt -alias company_root_cert \
  -file /path/to/your_company_root_cert.pem

sudo keytool -import -trustcacerts -keystore "/Applications/Jaspersoft Studio 7.0.1.app/Contents/Eclipse/features/jre.macosx.cocoa.x86_64.feature_17.0.8.1_1/eclipsetemurin_jre/Contents/Home/lib/security/cacerts" \
  -storepass changeit -noprompt -alias KASIKORNBANK \
  -file /Users/zhenyuan.h/Documents/KASIKORNBANK-ROOT-CA.pem

# 示例
sudo keytool -import -trustcacerts -keystore /Library/Java/JavaVirtualMachines/temurin-17.jdk/Contents/Home/lib/security/cacerts \
  -storepass changeit -noprompt -alias KASIKORNBANK \
  -file /Users/zhenyuan.h/Documents/KASIKORNBANK-ROOT-CA.pem

# 示例
sudo keytool -import -trustcacerts -keystore /opt/homebrew/Cellar/openjdk/24.0.2/libexec/openjdk.jdk/Contents/Home/lib/security/cacerts \
  -storepass changeit -noprompt -alias KASIKORNBANK \
  -file /Users/zhenyuan.h/Documents/KASIKORNBANK-ROOT-CA.pem
```

__解释命令参数__：

- `-import`: 导入证书

- `-trustcacerts`: 信任此证书作为证书颁发机构

- `-keystore`: 指定证书库路径

- `-storepass changeit`: 默认密码是 `changeit`

- `-noprompt`: 不提示确认

- `-alias`: 为证书指定别名

- `-file`: 要导入的证书文件
3. __验证证书是否导入成功__：

```bash
# 查看证书库中所有证书
keytool -list -keystore $JAVA_HOME/lib/security/cacerts -storepass changeit | grep -i kscm.kasikornbank.com

# 或查看特定证书
keytool -list -keystore $JAVA_HOME/lib/security/cacerts -storepass changeit -alias mycompany-root-cert
```
